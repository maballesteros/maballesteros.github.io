# Vocabulario

Aplicación React + Vite para practicar vocabulario básico de inglés a español.

## Scripts

- `npm install` – instala dependencias.
- `npm run dev` – arranca el servidor de desarrollo en `http://localhost:5173`.
- `npm run build` – valida tipos y genera la versión estática en `../../vocabulario` lista para publicarse en `/vocabulario/` del sitio.
- `npm run preview` – sirve la build generada.

La build se publica automáticamente en la carpeta raíz `vocabulario`, con `base` configurado para que funcione bajo `https://tu-sitio.com/vocabulario/`.
