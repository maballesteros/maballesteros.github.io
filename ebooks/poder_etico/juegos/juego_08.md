# Juego 8: Plantilla 3 Pasos Sprint

> **Objetivo**: Automatizar la estructura "CPS" (Contexto-Propuesta-Siguiente paso) para escribir mensajes de alto impacto bajo presión de tiempo.

## Jugadores y Roles
*   **Rol A (El Caos)**: Lanza un problema complejo y urgente.
*   **Rol B (El Ejecutivo)**: Tiene 1 minuto para escribir (en papel o móvil) un mensaje de respuesta perfecto usando la plantilla.

## Set-up Rápido: Crisis de Minuto

El **Caos** lanza una de estas bombas. El **Ejecutivo** tiene 1 minuto para escribir el CPS.

| Crisis (El Caos) | Contexto Real (Para el CPS) |
| :--- | :--- |
| "¡Se ha caído la web y estamos en Black Friday!" | Incidente crítico + Pérdida de ventas. |
| "El cliente dice que el logo rosa es horrible y amenaza con irse." | Insatisfacción estética + Riesgo de cancelación. |
| "Me he dejado las llaves de la oficina dentro y llega el CEO en 10 min." | Bloqueo de acceso + Visita VIP inminente. |
| "La presentación tiene mal los datos de facturación, ¡qué vergüenza!" | Error en entregable + Reputación dañada. |
| "No encuentro el archivo final y la imprenta cierra ya." | Pérdida de activo + Deadline crítico. |

## Mecánica
1.  Caos: "¡Oye, que el cliente dice que no le gusta nada y que si cambiamos la fecha, pero es que entonces no llegamos y qué hacemos!"
2.  Ejecutivo escribe:
    *   **Contexto**: Cliente insatisfecho + Riesgo de fecha.
    *   **Propuesta**: Reunión de urgencia con él para priorizar alcance.
    *   **Siguiente paso**: Yo le llamo ahora. Tú revisa el contrato.
3.  Se lee en voz alta. Si falta uno de los 3 pasos, pierde. Si es claro y directivo, gana.

## Debriefing (Preguntas de cierre)
*   ¿Te cuesta más definir el problema (Contexto) o proponer la solución (Propuesta)?
*   ¿Has incluido nombres en el Siguiente Paso? (Sin nombres, no hay responsabilidad).

### Variante
Hazlo con audios de WhatsApp. Tienes 20 segundos para grabar el CPS. Entrena la voz y la síntesis.
