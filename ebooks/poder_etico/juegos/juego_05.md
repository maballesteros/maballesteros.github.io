# Juego 5: No + Alternativa

> **Objetivo**: Perder el miedo a decir "No" eliminando la culpa, transformando la negativa en una contra-oferta constructiva.

## Jugadores y Roles
*   **Rol A (El Pedidor)**: Pide favores irracionales o inoportunos.
*   **Rol B (El Negociador)**: Dice no, pero ofrece algo viable.

## Set-up Rápido: Peticiones Trampa

El **Pedidor** elige una de estas "patatas calientes". El **Negociador** debe decir NO + Alternativa.

| Petición Irracional / Inoportuna | Ejemplo de Respuesta (No + Alternativa) |
| :--- | :--- |
| "¿Me prestas tu coche este finde?" | "No puedo prestártelo. Pero te llevo yo a la estación." |
| "Hazme este informe para mañana a las 8." | "No voy a poder tenerlo para esa hora. Pero el lunes a primera hora es tuyo." |
| "¿Me dejas dinero? Te lo devuelvo... pronto." | "No presto dinero a amigos. Pero te invito a comer hoy." |
| "Vente de fiesta que es jueves." | "Hoy no salgo. Pero el sábado me apunto a un café." |
| "Quédate con los niños tú solo toda la semana." | "Toda la semana no es viable. Pero repartimos los días: 3 tú y 4 yo." |
| "¿Me corriges esta tesis de 300 páginas hoy?" | "Leerme entera la tesis hoy es imposible. Pero puedo revisar la intro." |
| "Cámbiame el turno del sábado por favor..." | "Este sábado no puedo cambiarlo. Pero el del mes que viene sí." |
| "Dile a tu jefe que me contrate." | "No puedo enchufarte así. Pero te reviso el CV para que lo envíes." |

## Mecánica
1.  El Pedidor pide: "¿Me haces este informe para mañana?", "¿Me dejas el coche?", "¿Vienes a cenar hoy?".
2.  El Negociador debe usar la fórmula estrictamente:
    *   **NO**: "No puedo", "No voy a hacerlo", "Imposible". (Sin excusas).
    *   **+**: Pausa de 1 segundo.
    *   **ALTERNATIVA**: "Pero puedo mirarlo el lunes", "Pero te llevo yo", "Pero voy al café mañana".
3.  El objetivo es decir el NO con tono plano (sin subir agudos al final pidiendo perdón).

## Debriefing (Preguntas de cierre)
*   ¿Has sentido la tentación de explicar "por qué" no podías? (Esa es la trampa de la culpa).
*   ¿Cómo cambia la cara del otro cuando ofreces la alternativa?

### Pro Tip
Usa el "No" sándwich de validación: "Me halaga que me lo pidas (Pan) + No voy a poder (Carne) + Gracias por pensar en mí (Pan)".
