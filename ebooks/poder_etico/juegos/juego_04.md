# Juego 4: Broma o Descalificación

> **Objetivo**: Entrenar el "Detector de Agresión Encubierta" para distinguir rápidamente entre una broma amistosa y un ataque de estatus, y reaccionar proporcionalmente.

## Jugadores y Roles
*   **Rol A (El Guionista)**: Lee frases de una lista (algunas neutras, otras bordes).
*   **Rol B (El Detector)**: Clasifica y responde.

## Set-up Rápido: El Guion

El **Guionista** lee una de estas frases. El **Detector** debe gritar el color.

| Frase | Tono / Intención | Color Correcto |
| :--- | :--- | :--- |
| "Oye, qué bien te queda esa camisa." | Sincero / Cumplido | **VERDE** |
| "Uy, qué valiente viniendo así vestido..." | Sarcasmo / Juicio | **ROJO** |
| "¿Me pasas la sal, por favor?" | Petición neutra | **VERDE** |
| "A ver si aprendes a pasar la sal sin tirarla." | Paternalismo / Regañina | **ROJO** |
| "Te veo un poco cansado, ¿estás bien?" | Preocupación real | **VERDE** |
| "Tienes una cara horrible hoy, ¿no?" | Falsa preocupación / Ataque | **ROJO** |
| "Gracias por el informe, buen trabajo." | Agradecimiento | **VERDE** |
| "Por fin entregas algo decente." | Elogio con insulto implícito | **ROJO** |
| "¿Podemos hablar un momento?" | Propuesta de conversación | **VERDE** |
| "Tenemos que hablar (suspiro)..." | Generación de ansiedad | **ROJO** |

## Mecánica
1.  El Guionista dice una frase con tono ambiguo.
    *   Ej 1: "Vaya pelos traes hoy".
    *   Ej 2: "Menos mal que estás tú, Einstein".
2.  El Detector debe gritar **"VERDE"** (Amistoso) o **"ROJO"** (Ataque) en menos de 2 segundos.
    *   Si es VERDE: Sonríe y sigue.
    *   Si es ROJO: Lanza un meta-comentario neutro ("Ese comentario sobra", "¿A qué viene eso?", "Ignoro el tono").

## Debriefing (Preguntas de cierre)
*   ¿Es más difícil detectar o responder?
*   ¿Te ha dado miedo parecer "demasiado sensible" al marcar un ROJO?

### Variante
Juega con los ojos cerrados para guiarte solo por el tono de voz (paralenguaje).
