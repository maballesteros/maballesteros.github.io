# Juego 10: Anti-Teacher-Pupil

> **Objetivo**: Detectar cuando alguien te está dando lecciones no solicitadas (encuadrándote como alumno/ignorante) y restablecer el encuadre de igual a igual.

## Jugadores y Roles
*   **Rol A (El Maestro)**: Explica cosas obvias con tono condescendiente. "Mira, esto se hace así, déjame que te enseñe...".
*   **Rol B (El Par)**: Rompe el marco de alumno.

## Set-up Rápido: Lecciones del Maestro

El **Maestro** elige una frase para sentar cátedra. El **Par** debe romper el marco.

| Frase Condescendiente (Maestro) | Intención Oculta |
| :--- | :--- |
| "A ver, déjame que te esplique cómo va esto realmente." | Asumir que el otro no sabe. |
| "Lo estás haciendo bien, pero para tu nivel..." | Elogio tóxico (baja expectativa). |
| "Cuando lleves tant tiempo como yo aquí, entenderás..." | Argumento de antigüedad. |
| "No te preocupes, yo me encargo que es complicado." | Paternalismo / Inutilización. |
| "Es adorable que intentes hacerlo así." | Infantilización. |
| "Te voy a dar un consejo gratis..." | Consejo no pedido = Superioridad. |
| "¿Has leído el libro que te dije? Se nota que no." | Examen público. |

## Mecánica
1.  El Maestro empieza su sermón.
2.  El Par debe interrumpir suavemente usando técnicas de **Validación + Aporte Experto** o **Meta-comentario**:
    *   *Técnica 1 (Yo también sé)*: "Sí, conozco la teoría X. De hecho, mi enfoque aquí es Y por Z razon."
    *   *Técnica 2 (Cierre)*: "Entendido el concepto. Pasemos a la ejecución."
    *   *Técnica 3 (Ironía suave)*: "Gracias por la clase magistral, profe. Volvamos al tajo."
3.  Si B se queda callado asintiendo, pierde. Debe verbalizar su competencia.

## Debriefing (Preguntas de cierre)
*   ¿Notas cómo el "Maestro" se crece si no le paras?
*   ¿Qué frase te resulta más natural para decir "yo ya sé esto" sin ser arrogante?
